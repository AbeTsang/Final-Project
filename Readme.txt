Readme
Summary
This program simulates a server getting busy, and what it would when there is too much traffic. In the program, a queue is used to represent the server, and the program keeps enqueueing until the queueis fulled. When it is, the program asks the user if he/she wants to expand the server. There are also other useful functions in the program.

How to Run
1. Compile the program
2. Follow the menu

Dependencies
The program does not use any lirary on the web but the one written with the main program.

System requirements
This program could run both Windows and Linux. Not sure about Mac, as there was no implementation on it.
